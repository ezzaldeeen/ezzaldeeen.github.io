# ezzaldeeen.github.io

## Run locally

```
make run
```

Then open http://localhost:4000

## New post

```
make post title="Your Title"
```

Fill in the frontmatter fields:

```yaml
sub-title: ""   # subtitle shown below the title
overview: ""    # shown in the post header
abstract: ""    # shown in the homepage listing
```

## Publish a draft

```
make publish title="Your Title"
```

## Build

```
make build
```
